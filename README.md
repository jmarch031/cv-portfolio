# Portfolio Web3 premium - Julien Marchand
v8: thème clair, mise en page améliorée, fond unique animé compatible clair et sombre.

## Lancer en local
- Ouvre `index.html` dans un navigateur ou lance
  - `python3 -m http.server 8000` puis http://localhost:8000
  - `npx http-server .`

## Déploiement GitHub Pages
- Crée un repo et pousse les fichiers.
- Settings → Pages → Branche `main` → Dossier `root` → Save.
- Workflow `.github/workflows/deploy.yml` fourni.
