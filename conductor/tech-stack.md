# Technology Stack

## Core Frontend
- **Vite**: Modern frontend build tool for rapid development and optimized production bundling.
- **TypeScript**: Statically typed JavaScript for improved developer experience and code reliability.
- **HTML5**: Semantic document structure for a modern, accessible portfolio.
- **CSS3**: Modular responsive design using CSS Grid, Flexbox, and CSS Custom Properties (Variables) for theming.

## Interactive & Animation Engine
- **GSAP (GreenSock Animation Platform) v3**: High-performance animation engine for UI effects and timeline-based reveals.
- **ScrollTrigger**: Advanced scroll management synchronized with GSAP timelines.
- **Lenis**: Smooth scrolling library for a tactile, high-end feel.

## Deployment & Infrastructure
- **GitHub Actions**: Automated CI/CD pipeline for building and deploying the portfolio.
- **GitHub Pages**: Static site hosting with automated deployment from the `main` branch via GHA.

## External Resources
- **Google Fonts**: Custom typography using 'Plus Jakarta Sans' (Main) and 'Space Mono' (Tech/Accent).
- **Asset Hosting**: External CDNs and cloud storage for high-quality background videos and media.
