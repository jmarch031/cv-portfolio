# Technology Stack

## Core Frontend
- **HTML5**: Semantic document structure for a modern, accessible portfolio.
- **CSS3**: Responsive design using CSS Grid, Flexbox, and CSS Custom Properties (Variables) for theming.
- **JavaScript (ES6+)**: Custom application logic for theme switching, language management (i18n), and animations.

## Interactive & Animation Engine
- **GSAP (GreenSock Animation Platform) v3.12.5**: High-performance animation engine for UI effects and timeline-based reveals.
- **ScrollTrigger**: Advanced scroll management synchronized with GSAP timelines.
- **Lenis v1.0.34**: Smooth scrolling library for a tactile, high-end feel.

## External Resources & CDNs
- **Google Fonts**: Custom typography using 'Plus Jakarta Sans' (Main) and 'Space Mono' (Tech/Accent).
- **External CDNs**: Cloudflare and Unpkg for delivering GSAP and Lenis.
- **Cloud Hosting**: Assets like background videos and icons delivered from CodePen and Google services.
