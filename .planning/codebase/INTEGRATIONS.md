# Integrations

## CDN Services
- **cdnjs.cloudflare.com**: Serves core libraries like GSAP and its plugins (ScrollTrigger).
- **unpkg.com**: Delivers the Lenis smooth scroll library.
- **fonts.googleapis.com / fonts.gstatic.com**: Provides external typography assets.

## External Media & Assets
- **CodePen Assets**: Ambient background video hosted at `assets.codepen.io`.
- **Unsplash**: High-resolution photography for project covers and background effects.
- **Google Favicon Service**: Automatically retrieves site icons for the experience section.

## Social and Professional Platforms
- **LinkedIn**: Direct profile integration for professional networking.
- **Medium**: Integration of articles and publications via external linking.
- **Email**: Native `mailto` integration for direct communication.

## Key Internal Features
- **i18n**: Custom JavaScript implementation for French/English localization.
- **Theme Engine**: Data-attribute based dark/light mode toggle integrated with CSS variables.
